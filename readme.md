## Getting Started

First, run the development server:

```bash
npm install
```

```bash
npm start
```

#### Dependencies

- Native-Base
- React-navigation
  - native-stack
  - bottom-tabs
- FlatList
- SectionList
- Image-picker
- React-hook-forms
- Yup
- Axios
  - Interceptors (errors)
- AsyncStorage
- Auth workflow (JWT)

#### About

- Authentication
  - SingIn
  - SingOut
  - SingUp
  - Token + refresh token
- Reset Password
- Image upload
- Form control validation
- ContextApi
